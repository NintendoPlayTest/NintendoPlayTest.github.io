¿Qué es el Nintendo Switch Online: Playtest Program?

Tenemos previsto efectuar una prueba llamada Nintendo Switch Online: Playtest Program relacionada con una nueva función para el servicio Nintendo Switch Online de Nintendo Switch™.

Para ello, buscamos participantes para la prueba entre quienes dispongan de una Nintendo Switch.

Para participar, dichos usuarios deberán descargar en su consola Nintendo Switch un programa exclusivo.
Antes de inscribirte, consulta las reglas de solicitud.

Reglas de solicitud

Requisitos de solicitud

  • Disponer de una Nintendo Switch, Nintendo Switch Oled o Nintendo Switch Lite desde el jueves, 10-10-2024, 00:00 (CEST)
  • Disponer de una cuenta Nintendo registrada en uno de los siguientes países o regiones: Japón, Estados Unidos, Reino Unido, Francia, Alemania, Italia o España.

Duración del Playtest

Jueves, 24-10-2024, 03:00 (CEST) - Jueves, 01-1-2026, 01:59 (CET)
